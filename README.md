# README

Az alábi projekt a Debreceni Egyetem Szoftverfejlesztési Módszertanok nevű tárgy keretén belül készült. A projekt egy webes játékáruház elkészítéséből (Spring Boot keretrendszer használatával) és az áruház egy játékának implementálásából áll (Unity Engine-ben). Mindemellett a projekt backendje egy AWS szerveren fut a weben való eléréshez.


## Forráskódok:

Unity projekt forrásállományai: [Unity projekt](https://github.com/KonmaWasurenaide/SZFM_PROJEKT_2/tree/main/Assets)
Webáruház forrásállományai: [Webáruház projekt](https://github.com/KonmaWasurenaide/SZFM_PROJEKT_2/tree/main/WebDatabase)

## Elérés
A produktum bárki számára elérhető és megtekintehtő a következő linken keresztül: https://kusp.games 
A weboldalon regisztrációt és bejelentkezést követően lehetőség van a játékok online, böngészőből történő kiprobálására, továbbá letöltésére is.
